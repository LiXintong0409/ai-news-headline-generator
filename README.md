# AI News Headline Generator

Chinese news headline generation using fine-tuned BART.

## Overview

This project demonstrates the application of transformer-based models to automated news headline generation in Chinese. It showcases skills in natural language processing, deep learning, and computational journalism.

## Dataset

- **THUCNews** (Tsinghua University Chinese News)
- 740,000+ news articles across 14 categories
- Open-source, freely available for research use

## Tech Stack

- Python · PyTorch · Transformers (Hugging Face)
- BART-base-chinese (seq2seq model)
- ROUGE for evaluation
- Gradio for interactive demo
